## AI-Jobs Project
Интерактивное fullstack-приложение для поиска вакансий и анализа рынка труда в сфере искусственного интеллекта. 
Проект также позволяет оценивать приблизительные месячные накопления по каждой вакансии, сопоставляя зарплаты с индексами стоимости жизни и аренды по городам.

### Ключевые возможности
- Умный поиск: фильтрация вакансий по городам, категориям, годам и минимальной зарплате с расчетом потенциальных ежемесячных накоплений
- Интерактивная аналитика: динамические графики с распределением зарплат и вакансий по всему миру.
- Кэширование и сохранение состояния: фильтры не сбрасываются при переключении между страницами.

### Планы на будущее
В качестве дальнейшего развития проекта запланировано внедрение машинного обучения для следующих задач: 
1. Рекомендательная система релокации: предсказание наиболее выгодных локаций для переезда на основе баланса прогнозируемой зарплаты в ИИ-сфере, стоимости жизни, доступности вакансий по конкретному стеку и индекса чистых сбережений.
2. Прогнозирование зарплаты: предсказание дохода по конкретной вакансии, городу и навыкам на основе исторических трендов за 2025–2026 годы
